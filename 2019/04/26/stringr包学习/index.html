一.介绍

Stringr是由Hadley Wickham开发的一款专门针对字符串处理的R包。字符串作为一种数据类型充斥着我们的日常生活。不管你是数据分析师、市场专员、爬虫工程师也好，字符串是你日常工作当中接触非常多的一类数据。我们往往需要对字符串进行处理后，才能得到们想要的信息。这些处理包括字符串的查找、替换。

通过学习Stringr package 将具备一定的字符串处理能力，这对你日常的工作无疑是非常有效的。当然，字符串的处理所离不开一种非常强大的武器，那就是:正则表达式(regular expression)。

二.字符串基础

    #字符串长度
    install.packages("tidyverse")
    library(stringr)
    x <- c("a","i have pan")
    str_length(x)
    #str_length函数将告诉你字符串的长度

    #字符串合并
    z <- c("b","i have banana")
    str_c(x,z)
    a1 <- c("1")
    a2 <- c("2")
    str_c(a1,a2)
    str_c("pig","bird",sep=',')
    D <- c("GG",NA)
    str_c("|",str_replace_na(D),"|")
    #多个字符串转单个字符串
    str_c(c("x","y","z"),collapse = ",")

    #字符串提取
    chin <- c("一只猪","二头驴","三条狗")
    str_sub(a,1)
    str_sub(a,1,1)
    str_sub(a,-3,-2)

    #Locales
    str_to_lower和str_to_upper改变字符串大小写
    str_to_lower(c("XY","YY"))
    str_to_upper(c("XY","YY"),locale="tr")

    #基础匹配
    x<-c("java","C++","golang","JavaScript")
    str_view(x,"ava")

    #位置匹配
    ^ 匹配字符串的第一个字符
    $ 匹配字符串最后一个字符
    x <- c("apple","banana","pear")
    str_view(x,"^a")
    str_view(x,"a$")
    x <- c("apple pie","apple","apple cake")
    str_view(x,"apple")
    str_view(x,"^apple$")

    #
    \d :匹配任何的数值
    \s :匹配任何空白(tab分隔；空白)
    [abc]: 匹配a、b、c
    [^abc]：匹配除a、b、c
    #注意如果要匹配元字符本身，在R语言里转义是\\而非\

    #重复次数
    ？：0 or 1
    + :1 or more
    *：0 or more
    {n} 精确到n
    {n,} n或more
    {,m} 最多到m
    {n,m} n到m之间

更多的详细见stringr包的参考手册，及正则表达的相关书籍。
